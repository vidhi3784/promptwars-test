# 🧠 StudyAI Landing Page

Welcome to the **StudyAI** landing page project! This is a modern, visually stunning, and highly responsive landing page designed for an AI-powered educational platform.

## ✨ Key Features

*   **🎨 Modern Glassmorphism Design**: Utilizes translucent glass panels, ambient background glows, and a sleek dark theme for a premium feel.
*   **🚀 Hero Section**: Engaging headline with clear call-to-action buttons (Join Waitlist & Watch Demo) perfect for catching user interest.
*   **🎯 Feature Grid**: Three beautifully stylized cards highlighting core features:
    *   ⚡️ **Instant Summaries**: Quickly distill large documents.
    *   💬 **Socratic Tutor**: Interactive AI learning assistant.
    *   🎯 **Spaced Repetition**: Optimized knowledge retention.
*   **✉️ Waitlist Form**: Interactive email signup form with dynamic simulated loading states.
*   **🎉 Success Toast Notification**: Fully animated, slide-up success notification that appears upon successful form submission.
*   **📱 Fully Responsive**: Optimized to look great on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

*   **HTML5**: Semantic and accessible markup.
*   **Tailwind CSS (via CDN)**: Utility-first CSS framework for rapid, responsive, and maintainable styling.
*   **Vanilla JavaScript**: Handles form submission simulation, button loading states, and toast notification animations.
*   **Lucide Icons**: Beautiful, clean vector icons used throughout the design.
*   **Google Fonts**: Uses the modern and sleek ['Outfit'](https://fonts.google.com/specimen/Outfit) font family.

## 🚀 How to Run

Since this is a standalone HTML document utilizing CDNs, running it is incredibly simple and requires no build steps:

1.  **Clone or Download**: Ensure the `studyai.html` file is saved to your local machine.
2.  **Direct File Open**: Simply double-click the `studyai.html` file. It will automatically open and run in your default web browser (e.g., Chrome, Firefox, Edge, Safari).
3.  **Local Development Server (Recommended for Extensions)**: If you plan to extend this project with local assets, it's best to run a local server. For example, if you have Python installed:
    ```bash
    # Run this in your terminal within the project folder
    python -m http.server 8000
    ```
    Then, navigate to `http://localhost:8000/studyai.html` in your web browser.
