# 🏥 Health-Check Dashboard

A lightweight, modern, and interactive Smart Vitals Monitor built as a single-file application. It allows users to input their age and heart rate, evaluates the inputs, and instantly generates a simulated health report with a dynamic user interface.

## ✨ Key Features

- **❤️ Dynamic Vitals Alert:** Automatically transitions the entire application theme to a critical red alert state if a resting heart rate over 100 BPM is detected.
- **🩺 Simulated Diagnostics:** Generates random, actionable health tips from a curated database of advice.
- **💎 Premium Medical UI:** Designed with a clean, trustworthy aesthetic featuring modern soft shadows, rounded elements, and high-quality typography.
- **⚡ Ultra-Lightweight:** The entire application is under 10KB, utilizing a single-file architecture with zero build steps required.
- **📱 Fully Responsive:** Carefully constructed layout that works seamlessly across desktops, tablets, and mobile devices.

## 🛠️ Tech Stack

- **HTML5:** Core semantic structure and layout.
- **Tailwind CSS (CDN):** Rapid, responsive utility-first styling.
- **Vanilla JavaScript:** Powers the conditional health logic, randomized report generation, and dynamic DOM manipulation.
- **Google Fonts (Outfit):** Provides a clean, highly legible, and premium modern look suitable for health applications.

## 🚀 How to Run

Because this application uses a single-file architecture with no heavy dependencies or node modules, running it is incredibly simple:

1. **Locate the file:** Find `health-dashboard.html` in your project directory.
2. **Open in Browser:** Simply double-click the file, or drag and drop it into any modern web browser (Chrome, Firefox, Safari, Edge).
3. **Start Monitoring:** Enter an Age and a Heart Rate, then click **"Simulate Report"** to see the logic in action. 
   > 💡 *Tip: Try entering a heart rate above 100 to see the emergency alert mode!*

No `npm install`, no local development servers required—just plug and play!
